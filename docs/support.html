Вопросы и ошибки также можно оставить в GitHub Issues.

Как удалить все данные

Откройте Приложение → вкладка «Настройки» → раздел «Данные» → «Удалить данные» → подтвердите. Все записи будут стёрты с устройства безвозвратно. Данные также удаляются при удалении Приложения.

Как восстановить покупки

Главный экран → «Луна Premium» (или Настройки) → на экране подписки нажмите «Восстановить покупки», войдя в тот же аккаунт магазина приложений, с которого оформляли подписку.

Частые вопросы

Почему прогноз не совпал с реальной датой?

Прогноз ориентировочный (обычно ±2 дня) и уточняется по мере накопления ваших данных. На цикл влияют стресс, болезни, поездки и другие факторы.

Мои данные кто-нибудь видит?

Нет. Всё хранится только на вашем устройстве, аккаунт не нужен. Подробнее — в Политике конфиденциальности.

Луна — это врач?

Нет. Луна — ИИ-собеседница для поддержки. Она не ставит диагнозы и не советует лекарства. По медицинским вопросам обращайтесь к врачу.

Придут ли напоминания о таблетках?

В версии 0.1.0 настройки напоминаний сохраняются, а сами push-уведомления появятся в ближайшем обновлении.
